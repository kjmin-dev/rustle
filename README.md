# Rustle

# Getting Started

### Setup node.js
```sh
nvm install # Install node.js version defined in .nvmrc
```
### Run prepare: execute prepare script for each repos
```sh
yarn prepare
```
### Run dev: run all repos
```sh
yarn dev
```

### Run build: build all repos
```sh
yarn build
```

### Create new repo
```sh
yarn new
```
